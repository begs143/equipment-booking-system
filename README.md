# TDCI-IT Equipment Booking System 🖥️📅

A specialized reservation platform for IT assets. This system allows users to view, book, and track equipment while providing administrators with oversight of institutional resources.

## ✨ Core Features
* **Live Equipment Catalog:** Browse available IT hardware with real-time status updates.
* **Conflict-Free Booking:** Backend validation ensures no two users can book the same item at the same time.
* **Approval Workflow:** Integrated status management (Pending, Approved, Returned, Overdue).
* **Automated Logging:** Tracks the history of every piece of equipment for accountability.

## 🛠️ Technical Implementation
* **Backend:** Laravel 10 (Eloquent, Migrations, Controllers)
* **Frontend:** Clean UI using Blade and Bootstrap
* **Logic:** Custom validation rules for date/time overlap prevention.
